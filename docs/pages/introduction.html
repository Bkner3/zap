Introduction

Welcome to the Zap PM documentation!

It is designed to be lightweight, fast, and easy to use. With Zap, you can easily install, update, and manage packages.

Zap is a command-line package manager written in Python. The source code is available on GitHub.

My package manager can be used either as a Python script or as a standalone executable.
If you want to use it as a Python script you need to have Python 3.x installed on your system and the python libraries requests tqdm colorama Click here to install the required libraries. If you want to use it as a standalone executable, you can download the latest release from the GitHub Releases page.

# Zippy Asset Packager
`v0.6.0-beta`

## Introduction

Welcome to the Zap PM documentation! It is designed to be lightweight, fast, and easy to use. With Zap, you can easily install, update, and manage packages.

Zap is a command-line package manager written in Python. The source code is available on [GitHub](https://github.com/Bkner3/zap).

Zap can be used either as a Python script or as a standalone executable.

### Requirements & Installation

* **As a Python script:** You need to have [Python](https://www.python.org) 3.x installed on your system along with the following libraries: `requests`, `tqdm`, and `colorama`.
  
  Run this command to install the required dependencies:
  ```bash
  pip install requests tqdm colorama

* **AS a standalone executable:** To install as a standalone executable you can use the commad below in powershell

  ``` powershell
    irm https://raw.githubusercontent.com/Bkner3/zap/refs/heads/main/setup/install.ps1 | iex

After instaling reset your terminal and type zap help for the command list.

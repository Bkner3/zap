01 How to install

Run the following command in your terminal:

Windows Powershell
irm https://raw.githubusercontent.com/Bkner3/zap/refs/heads/main/setup/install.ps1 | iex

02 commands

zap install <pkg>install a package
zap update <pkg>update all packages
zap remove <pkg>remove a package
zap download <pkg>download without installing
zap upgradeupgrade zap
zap add <repo>add a new repository
zap listlist all installed packages
zap info <pkg>show package details
zap helpshow help message
zap configconfigure zap settings

03 how it works

1

update

Downloads index.zip from each repository and extracts the package list.

2

search

Looks up the package name in the index and saves all package information.

3

download

Downloads the .zip from the URL using multithreading.

4

install

Moves the zip to the final folder, extracts it, adds to the package database, and removes the archive.
